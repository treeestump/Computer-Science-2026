Summary

This program manages student grades using a pickle file for persistence. It prompts for three student names, loads existing grades from a file, then enters a loop where users can view a student's grades and average, optionally modify grades, and choose to continue or exit. Grades are saved to a binary pickle file whenever changes are made.
