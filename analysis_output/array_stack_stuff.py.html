Summary

This code creates a new list by interleaving elements from two input lists based on index position. At even indices, it takes elements from the second list; at odd indices, it takes elements from the first list. It handles lists of different lengths by continuing until the longer list is exhausted.

INITIALIZE l1 = [3, 6, 9, 12, 15, 18, 21]
INITIALIZE l2 = [4, 8, 12, 16, 20, 24, 28]
INITIALIZE l3 = []

FUNCTION makeyippeelist(one, two):
    CREATE empty list yippee
    SET done = False
    SET indexnum = 0
    
    WHILE done is False:
        TRY:
            IF indexnum is even:
                APPEND two[indexnum] to yippee
            ELSE:
                APPEND one[indexnum] to yippee
            INCREMENT indexnum
        
        CATCH IndexError:
            IF length of one > length of two:
                IF indexnum > length of one - 1:
                    SET done = True
            ELSE:
                IF indexnum > length of two - 1:
                    SET done = True
            INCREMENT indexnum
    
    RETURN yippee
