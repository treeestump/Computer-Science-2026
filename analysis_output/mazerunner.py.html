Summary

This program implements a breadth-first search (BFS) maze solver. It reads a maze from a text file where '#' represents walls and 'E' represents the exit, then explores the maze starting from position (1,1), tracking visited cells and their step numbers until it finds the exit.

Algorithm pseudocode

LOAD_MAZE(filename):
    Open file and read each line
    For each line, convert characters to list (excluding newlines)
    Return 2D grid of characters

SEARCH_MAP(map):
    Initialize empty queue (next_up)
    Set starting position to (1, 1)
    Initialize step counter to 1
    Initialize empty dictionary for searched positions
    
    WHILE current position is not 'E':
        FOR each direction (up, down, left, right):
            IF neighbor is not wall AND not already searched:
                Add neighbor to queue
            ELSE IF neighbor was searched with lower step count:
                Update step to neighbor's step + 1
        
        Mark current position as searched with current step
        Increment step counter
        Pop next position from front of queue
        Update current position and coordinates
        Print current state
    
    Print all searched positions with their step numbers

MAIN:
    Load maze from 'maze_runner.txt'
    Run search_map on loaded maze
    Wait for user input
